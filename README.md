# Archive Reminder

Remind what you read before (1 week ago).

## Dependency

- Supabase
  - [Auth](https://supabase.com/docs/guides/auth/overview)
  - [Database](https://supabase.com/docs/guides/database)
- direnv

### Usage

Start the project:

```
deno task start
```

This will watch the project directory and restart as necessary.
